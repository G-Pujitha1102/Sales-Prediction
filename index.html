üìå Sales Prediction ‚Äì Machine Learning Web App

This project predicts future sales using Machine Learning models ‚Äì Linear Regression and Random Forest Regressor ‚Äì based on advertising datasets. It includes data preprocessing, model training, evaluation metrics, and visualizations in a Streamlit web app, making it interactive and accessible online.

üîó Live Demo | üîó GitHub Repository

üéØ Objective

Analyze historical advertising data through Exploratory Data Analysis (EDA), uncover patterns between TV, Radio, and Newspaper budgets and sales. Enable users to input budgets and predict sales using Linear Regression and Random Forest models. Classify sales as Low, Average, or High, visually compare predictions through bar charts, and indicate the most accurate model for data-driven marketing and business decisions.

üß™ Methodology

  1. Data Loading & Exploration
    • Loaded dataset with advertising budgets and sales data.
    • Explored summary statistics and distributions.
  2. Data Splitting
    • The dataset was divided into training and testing subsets (typically 80:20) to ensure unbiased evaluation.
    • Models were trained exclusively on the training data and validated on the test set.
  3. Model Training
    • Built Linear Regression and Random Forest models to predict sales.
    • Trained models on the training data features (TV, Radio, Newspaper budgets).
  4. Model Evaluation
    • Calculated metrics: R¬≤ Score, RMSE (Root Mean Squared Error), MAE (Mean Absolute Error).
    • Compared models based on these metrics.
  5. Visualization & Prediction
    • Interactive web app to input advertising budgets and get sales predictions.
    • Visual comparisons of model predictions and EDA plots.

üöÄ How to Run

  1. Clone the repo:
    git clone https://github.com/G-Pujitha1102/Sales-Prediction.git
  2. Navigate to project folder:
    cd Sales-Prediction
  3. Create and activate a virtual environment (optional but recommended):
    python -m venv venv
    source venv/scripts/activate
  4. Install dependencies:
    pip install -r requirements.txt
  5. Run the Streamlit app:
    streamlit run app.py
